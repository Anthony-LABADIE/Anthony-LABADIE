# 💫 About Me

Développeur Web Full Stack basé à Bordeaux 🇫🇷

En CDI chez **JDC SA**, je travaille sur une suite d'applications web multiplateforme : backoffice métier, tableaux de bord, rapports et exports de données (PDF, CSV, Excel).

J'accorde une attention particulière à la qualité du code, la réutilisabilité des composants et la documentation.
